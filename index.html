Voice Gateway

A VoIP Platform for "Packetizing the Edge of the PSTN Network". With on-board V5.2, ISDN-PRI or GR303 and optional MGCP/SIP, the platform can handle up to 480 channels of voice/fax. VG4800IP EdgeBlade™ is a high-density, integrated DSP platform ideal for developing a variety of VoIP access and core equipment. VG4800 supports LBR and PCM codecs.

Media Gateway

A powerful Single Board Media Gateway platform with MGCP module designed for TDM and IP based applications. MG4800IP can be used to build Access Gateways, Tandem Media Gateways, Wireless to Wireline Gateways, IP to PSTN Trunking Gateways and Remote DLCs. MG4800IP supports V5.2, PRI and GR303 protocols.

Systems

Signal Conversion Systems-Convertix™ : Helps interconnect PSTN networks with Access equipment on different signaling systems. Solutions available for V5.1AN-V5.2LE, V5.1LE-V5.2AN, CAS/MF to V5.x, PRI to V5.x on a choice of 16, 48 and 64 port systems.

Signal Grooming & Monitoring - Condor GroomET: T1/E1 signal monitoring and grooming systems, which taps live T1/E1 streams, switches and grooms the signaling channels. Available in a choice of 16, 48 and 64 port systems.
