MultiLink-PPP

DC4800PP is a MultiLink PPP based data communication platform implemented as a CompactPCI module, capable of generating and processing PPP/ML-PPP traffic on up to 16 T1/E1 interfaces simultabneously and a 100 mbps fast Ethernet connection. 16 E1 connections fully utilized can offer WAN connection of 32 mbps. The fully configurable on-board software is a scalable s implementation of the IETF-defined family of protocols: Point-to-Point Protocol (PPP), MultiLink PPP (ML-PPP) Protocol, Bandwidth Allocation Control Protocol (BAP/BACP) and related standards. The module can also be configured with CoVPN software to implement end to end VPN connectivity. Sample host applications for a variety of operating systems including Linux and RTOS are provided in source code to aid quick integration.
